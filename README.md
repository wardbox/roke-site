# roke.dev

Marketing site for [Roke](https://github.com/wardbox/roke), a Wasp starter template with sensible defaults.

## Development

```bash
pnpm install
pnpm dev
```

Site runs at http://localhost:4321

## Stack

- [Astro](https://astro.build)
- [Tailwind CSS v4](https://tailwindcss.com)
- [astro-icon](https://github.com/natemoo-re/astro-icon) with Phosphor Icons

## Deployment

```bash
pnpm build
```

Static output in `./dist/`
